# E-Commerce Sales Data Analysis

This project analyzes e-commerce sales data using Python for data cleaning, exploratory data analysis, and visualization.

## Project Goals
- Clean raw data
- Handle missing values
- Remove duplicate records
- Detect outliers
- Perform exploratory data analysis (EDA)
- Visualize insights
- Generate business recommendations

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

## Files Included
- `e_ticaret_proje.ipynb` → Main analysis notebook
- Cleaned CSV datasets
- HTML visualization outputs

## Key Insights
- Best-selling products identified
- Revenue distribution analyzed
- Customer behavior patterns examined
- Data quality issues resolved

This project was completed as part of a learning process and improved for portfolio purposes.
